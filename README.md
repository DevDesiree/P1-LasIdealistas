## Guía de Instalación

1. Clona este repositorio en tu máquina local:

   ```bash
   git clone https://github.com/DevDesiree/P1-LasIdealistas.git
    ```

2. Abre el archivo index.html en tu navegador web.

3. ¡Listo! Puedes comenzar a explorar el proyecto.
